# README
This repository is for the ENGG2112 project. We are making machine learning models for seizure detection

This repository was originally a branch of the [Github](https://github.com/biomedepi/seizeit2) used to process all imaging modalities included in the [SeizeIT2 dataset](https://openneuro.org/datasets/ds005873).
